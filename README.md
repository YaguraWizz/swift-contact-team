# Swift Contact Team

Проект на Swift, реализующий телефонную книгу с использованием Core Data. Приложение позволяет создавать, редактировать, удалять и просматривать контакты с сохранением данных.

## Основные возможности
- Хранение данных в Core Data.
- Просмотр, добавление, редактирование и удаление контактов.
- Фильтрация контактов по имени и статусу избранного.
- Сортировка контактов по различным критериям.
- Асинхронная оптимизация операций сохранения и удаления данных.

## Установка

### Требования
- Xcode 13 или выше.
- iOS 15 или выше.

### Шаги установки
1. Скачайте или клонируйте репозиторий.
2. Откройте проект в Xcode.
4. Соберите и запустите проект на симуляторе или устройстве.

## Структура проекта

### Модель данных
- **Contact**: Модель Core Data с полями:
  - `name`: имя контакта (String).
  - `birthdate`: дата рождения (Date).
  - `phoneNumber`: номер телефона (String).
  - `email`: электронная почта (String).
  - `isFavorite`: статус избранного (Bool).
  - `notes`: заметки (String).

### Основные экраны
- **ContentView**: Главный экран приложения.
- **ContactDetailView**: Экран отображения подробной информации о контакте.
- **NoContactView**: Экран, отображаемый при отсутствии контактов.
- **ContactCreationView**: Экран создания нового контакта.
- **ContactEditView**: Экран редактирования существующего контакта.

### Дополнительные функции
- **Контекст Core Data**: Используется для управления объектами и изоляции изменений перед их сохранением.
- **Асинхронные операции**: Оптимизация удаления данных через фоновую задачу для улучшения производительности.

## Тестирование
1. Запустите приложение.
2. Добавьте несколько контактов через экран создания.
3. Проверьте отображение деталей контакта при нажатии.
4. Попробуйте редактировать и удалять контакты.
5. Используйте фильтрацию и сортировку для проверки дополнительных функций.
6. Перезапустите приложение, чтобы убедиться в сохранении данных.

## Пример использования
1. **Добавление контакта**: Нажмите кнопку добавления, заполните поля и сохраните.
2. **Редактирование контакта**: Cмахните влево и выберите "Редактировать".
3. **Удаление контакта**: Смахните влево на строке контакта и выберите "Удалить".

## Заключение
Core Data предоставляет мощный инструмент для управления локальными данными в iOS-приложениях. Проект демонстрирует удобную реализацию CRUD-операций, сортировки и фильтрации данных.
